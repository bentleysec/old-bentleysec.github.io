WebGoat

Introduction

Webgoat is a project by OWASP. It is an insecure web application to that is useful to practice different web application vulnerabilities.

Installation

Installation is very straightforward. You need to have java installed on your machine. Download the jar file here.

Start the server:

java -jar webgoat-server-8.0.0.VERSION.jar

By default WebGoat starts on port 8080 you can easily change it to be port 80 by specifying –server.port=80. I am hosting WebGoat in my homelab so I wanted to host the server on my LAN. You can easily change the address of the server by specifying –server.address=(address).

Once the server is running you can navigate to (address):port/WebGoat. You will be greeted by this page:

Create an account and then you can start going through the lessons.
